# Multi-Branch Inventory Management System

## Project Overview

The **Multi-Branch Inventory Management System** is a backend application developed using **ASP.NET Core Web API**.
The system enables businesses to manage inventory across multiple branches, track stock levels, handle stock transfers, record sales and purchases, and generate reports.

The application follows **Agile Scrum methodology**, where development is divided into **7 sprints**, each implementing a set of features.

This project demonstrates concepts such as **Object-Oriented Programming, Web API development, Entity Framework Core, JWT Authentication, and Role-Based Authorization**.
